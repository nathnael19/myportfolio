# 🚀 Modern Developer Portfolio

A sleek, premium personal portfolio website built with modern web technologies. This project features a high-performance architecture, stunning animations, and an integrated AI assistant.
## ✨ Features

- **🤖 AI Chatbot Assistant**: Integrated Google Gemini AI to answer questions about skills, experience, and projects in real-time.
- **📱 Fully Responsive**: Optimized for all devices, from desktop to mobile, with a custom responsive navigation system.
- **🌗 Dark Mode**: Support for system-based and manual theme toggling with smooth transitions.
- **⚡ Performance First**: Built with Vite and React for near-instant load times and rapid state updates.
- **🎥 Premium Animations**: Powered by Framer Motion for smooth, high-quality interactions and staggered entry effects.
- **📂 Project Showcasing**: Dynamic project grid with "Show More/Less" functionality and category-based filtering.
- **📧 Contact Form**: Modern, interactive contact form with real-time feedback.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **AI Backend**: [Google Gemini Pro API](https://ai.google.dev/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository:**

   ```bash
   git clone <your-repo-url>
   cd modern-developer-portfolio
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env` file in the root directory and add your Gemini API key:

   ```env
   VITE_GEMINI_API_KEY="your_api_key_here"
   ```

4. **Launch the development server:**
   ```bash
   npm run dev
   ```
